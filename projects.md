# Projects

以下以「成果導向」呈現：背景 → 你做了什麼 → 量化成果 → 技術堆疊。

---

## 1) 專案名稱：XXX（例：LINE 客戶分眾推播與 CRM 流程整合）
- **問題**：原流程需要人工整理名單、訊息無法個人化、追蹤困難  
- **解法**：建立資料欄位規則與分眾標籤、串接自動推播流程、建立回流追蹤  
- **成果**：例如「推播效率提升 3 倍」「錯誤率下降 70%」  
- **技術**：Node.js / Python / LINE Messaging API / Google Sheets / Webhook

## 2) 專案名稱：XXX（例：檔案治理與 NAS 同步策略）
